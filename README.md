# 웹 서비스 소개
수어를 알파벳으로 번역해주는 기능을 갖춘 AI ChatGPT입니다. 

# 주요 기능
1. 번역이 필요한 사진 첨부
2. 첨부한 사진이 어떤 뜻인지 확인

   
# 🛠️사용tool
- VSCode
- Django
- MLflow
- AWS
- ChatGPT API
